#DOM 标签组件
virtual DOM模型涵盖了几乎所有的原生DOM标签，virtual DOM如同隔离的沙盒，React处理并不是直接操作和污染原生DOM，这样不仅保持了性能的高效和稳定， 而且降低了直接操作原生DOM而导致错误的风险。
